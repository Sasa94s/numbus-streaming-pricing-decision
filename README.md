# Price It Right — Nimbus Streaming Pricing Decision

This repository contains the starter files for the *Price It Right* project, the
capstone of Course 4 in Udacity's Applied Statistics Nanodegree.

You're a Decision Scientist at Nimbus Streaming, asked to recommend whether to
raise the Standard tier from \$12.99 to \$14.99. The project brief, setup
instructions, and grading rubric live in your Udacity Classroom.

## Repo layout

```
.
├── data/                     # five input CSVs (already provided)
└── starter/
    ├── nimbus_decision_starter.ipynb   # the notebook you fill in
    └── decision_memo_template.md       # memo scaffold (a .docx version is also provided)
```

## Deliverables

- `nimbus_decision_starter.ipynb` (filled in)
- `decision_memo.md` or `decision_memo.docx`
- `assumptions_and_sources.md` (filled in)
